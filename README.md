# jspsych-vwp-six-times

In this experiment participant take lessons about the German language. It is to
be expected that they learn from these lessons. This is measured using
a Visual World Paradigm, where understanding the German grammar might affect the
looking behavior.

As the participants are learning, there are 4 different lists in which they engage.
Where the sounds and images are setup up differently, in order to minimize
learning effects.

## Client

This version of a visual world paradigm has been made for Dorus Berkers phd student
at the ILS(labs) of the UU.
